# AI Sports Injury Detection - Frontend

## Overview

This is the frontend application for the **AI Sports Injury Detection** project developed as part of the Infosys Internship.

The application provides an intuitive user interface for:

- User Authentication
- Athlete Profile Management
- Dashboard
- Future Injury Detection Results
- AI Prediction Visualization

The frontend is developed using **React.js**.

---

## Tech Stack

- React.js
- React Router DOM
- Axios
- CSS

## Installation

Clone the repository

```bash
git clone <repository-url>
```

Move to frontend directory

```bash
cd frontend
```

Install dependencies

```bash
npm install
```

---

## Running the Application

Start the development server

```bash
npm run dev
```

Application runs at

```
http://localhost:5173
```

---

## Backend Connection

The frontend communicates with the backend using REST APIs.

Example API URL

```
http://localhost:5000/api
```

Axios is used for making API requests.
