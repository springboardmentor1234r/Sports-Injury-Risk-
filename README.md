# Sports-Injury-Risk-Detection-System
## Project Description
The Sports Injury Risk Detection System is a web application developed using FastAPI. It provides user authentication and player management as the foundation for future AI based sports injury prediction.
## Features
- User Registration
- User Login
- Player Management
- FastAPI Backend
- HTML, CSS,JavaScript Frontend
- SQLite Database
- Swagger API Documentation
## Technologies Used
- python
- FastAPI
- SQLAlchemy
- SQLite
- HTML
- CSS
- JavaScript
## How to Run
1. Install dependencies:
   pip install -r requirements.txt
2. Start the server:
   uvicorn main:app --reload
3. Open Swagger:
4. http://127.0.0.1:8000/docs
