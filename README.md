# Sports Injury Risk Detection

## Project Overview

This project is being developed as part of the Infosys Springboard Virtual Internship.

The objective is to build an AI-powered backend that helps detect the risk of sports injuries using video analysis and machine learning techniques.

## Technologies Used

- Python
- FastAPI
- Uvicorn
- Pydantic
- Git
- GitHub

backend/
